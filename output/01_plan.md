# 웹기획자 — arsen-ai 메인 홈페이지 기획

## 사이트 목적
AI 에이전트 플랫폼 arsen-ai의 공식 홈페이지.
방문자가 "이게 뭔 프로젝트야?"에서 "나도 써봐야겠다"로 전환되는 랜딩 페이지.

## 타겟
- AI 도구에 관심 있는 개발자/학습자
- 유튜브로 유입된 일반 사용자
- AI 스터디 모임 잠재 참여자

## 페이지 구조 (단일 스크롤)

### 1. 네비게이션 (고정)
- 로고: ARSEN AI
- 메뉴: 프로젝트 / 에이전트 / 채널 / 연락처
- 우측: GitHub 버튼

### 2. 히어로 섹션
- 메인 타이틀: "AI 에이전트로 일하는 방식을 바꿉니다"
- 서브: 타이핑 애니메이션으로 프로젝트명 순환
  (arsen-dashboard → ai-tools → sns-ops → member-system)
- CTA 버튼 2개: "프로젝트 보기" / "AI 모임 신청"
- 배경: 파티클/그리드 애니메이션

### 3. 프로젝트 섹션
카드 4개:
| 프로젝트 | 설명 |
|---------|------|
| arsen-dashboard | AI 에이전트 통합 대시보드 |
| ai-tools | AI 도구 자동화 모음 |
| sns-ops | SNS 운영 자동화 |
| member-system | AI 모임 신청자 관리 시스템 |

### 4. 에이전트 팀 섹션
아이콘 그리드로 팀 소개:
- 메타팀 (검증/재검증)
- 코드팀 (리뷰/수정)
- 웹빌더팀 (기획/개발)
- 전략팀 (분석/리포트)

### 5. 유튜브 채널 섹션
- 채널 소개 + 구독 CTA
- "AI를 배우는 가장 빠른 방법"

## UX 흐름
방문 → 히어로에서 브랜드 인식 → 프로젝트로 신뢰 형성 → 에이전트팀으로 기술력 어필 → 유튜브로 커뮤니티 유입 → CTA 클릭
